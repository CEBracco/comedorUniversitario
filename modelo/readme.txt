El ejemplo presentado realiza la carga de registros y el armado completo del modelo de la aplicacion
Para visualizar los resultados del ejemplo se debe disponer de un cliente de base de datos como mysql workbench o phpMyAdmin

los pasos para ejecutar el ejemplo son los siguientes:

-Crear un nuevo usuario de mysql con username: 'ttps' y password: 'ttps2015' (como alternativa modificar el archivo persistense.xml)
-Crear un nuevo esquema llamado 'comedor'
-Ejecutar el servlet 'TestComedor.java' ubicado en el paquete 'test'
-Examinar los resultados en la base de datos 'comedor'
